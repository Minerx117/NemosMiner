NemosMiner API (version 0.2.0.0)

API data is available at http://localhost:[API port]/<resource>. (The default API port is 3999)

File and Scripts

Anything other than the resources listed below will be looked for in the web directory and sent if it exists.

Any .ps1 files in the web/scripts directory will be run and their output sent. They get passed a $Parameters object with keys and values from the URL string. Test it with this demo: /scripts/demo.ps1?message=Hello%20world! This feature can be used to gather data that the API doesn't expose directly.

Resources

APIVersion

/apiversion

Returns the version number of NemosMiner API.

BTC Exchange rate

/btcratefirstcurrency

Shows the curent BCT exchange rate.

BrainJobs

/brainjobs

Returns the brainjobs information.

CompareMiners

/compareminers

Returns the Variables.compareminers information.

ComparePools

/comparepools

Returns the Variables. comparepools information.

Config

/config

Shows the full configuration for the running script, after all variable substitution.

Devices

/devices

Shows information about all mining devices.

Enabled Devices

/devices/enabled

Shows information about all enabled mining devices.

Disabled Devices

/devices/disabled

Shows information about all disabled mining devices.

Unsupported Devices

/devices/unsupported

Shows information about all unsupported mining devices.

Earnings

/earnings

Returns the earnings data.

EarningsPool

/earningspool

Returns the earnings pool information.

EarningsTrackerJobs

/earningstrackerjobs

Returns the earningstrackerjobs information.

First currency

/firstcurrency

Returns the firstcurrency information.

Miners

/miners

Shows all available miners data, use for debugging.

Best Miners

/miners/best

Shows the best miners data, use for debugging.

Enabled Miners

/miners/enabled

Shows the enabled miners data, use for debugging.

Disabled Miners

/miners/disabled

Shows disabled miners data, use for debugging.

Failed Miners

/miners/failed

Shows the failed miners data, use for debugging.

Idle Miners

/miners/idle

Shows the idle miners data, use for debugging.

Running Miners

/miners/running

Shows the running miners data, use for debugging.

Mining cost

/miningcost

Shows current mining cost.

Mining earnings

/miningearning

Shows current mining earning.

Mining profit

/profit

Shows current mining profit.

PoolsConfig

/poolsconfig

Shows the pool configuration entries.

Pools

/pools

Shows all pool data entries.

Best Pools

/pools/best

Shows the best pool configuration entries.

Enabled Pools

/pools/enabled

Shows the enabled pool configuration entries.

Disabled Pools

/pools/disabled

Shows the disabled pool configuration entries.

Rates

/rates

Shows exchange rates (currently only BTC), use for debugging.

Stats

/stats

Shows all stat data in the $Stats hashtable, use for debugging.
